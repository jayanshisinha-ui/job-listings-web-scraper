# 📊 Job Listings Web Scraper Using Python

---

## 📌 Project Overview

> This project is a **Python-based web scraping application** that extracts job listing data from a website and stores it into a structured **CSV file**.

The scraper automatically collects:

- **Job Title**
- **Company Name**
- **Location**
- **Scraped Date**

This project demonstrates how to automate **data extraction from websites** using Python.

---

## 🎯 Objectives

- [x] Understand the concept of **Web Scraping**
- [x] Extract structured data from HTML pages
- [x] Store extracted data into CSV format
- [x] Automate data collection using Python

---

## 🛠️ Technologies Used

- **Python**
